# ⏳ Historia del Desarrollo Web

El desarrollo web ha evolucionado drásticamente desde sus inicios.

## Orígenes (1991)

El primer sitio web fue lanzado por **Tim Berners-Lee** en el CERN. En esta época, las páginas eran estáticas y se enfocaban en compartir información.

## Web 2.0 (Mediados de 2000)

Esta era trajo la interactividad, redes sociales y aplicaciones dinámicas. Se popularizaron tecnologías como AJAX.

## Tendencias Actuales

* **Single Page Applications (SPA):** Aplicaciones que cargan una sola página HTML y el contenido se renderiza dinámicamente con JavaScript (usando frameworks como React o Vue).
* **Mobile First:** Diseñar pensando primero en la experiencia móvil.
* **APIs y Microservicios:** Descomposición de aplicaciones en servicios más pequeños e independientes.
